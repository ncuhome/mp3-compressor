# mp3 compress

## Guide

### Clone

```bash
git clone https://github.com/ncuhome/mp3-compresser
```

### Installation

```bash
npm i
# or
yarn
# or
pnpm i
```

Install [ffmpeg](https://ffmpeg.org/)

### Usage

Put your mp3 files to ./input/ folder

```bash
pnpm start
```

Then will output the compressed files to ./output/ folder
